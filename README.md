#Client Portfolio Website

A modern, responsive portfolio website developed for a client to showcase their professional profile, skills, services, projects, and contact information.

✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean UI
- Smooth scrolling navigation
- Hero section with call-to-action
- About section
- Services section
- Portfolio/Projects showcase
- Skills section
- Contact section
- Optimized for performance and accessibility

🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
